Y Combinator live scrapper

I developed YC News Grabber as a web app that collects and sorts articles from Y Combinator's news section in real-time. I used Python and BeautifulSoup for scraping, Pandas for data handling, and Flask with Bootstrap to create a simple interface. My app allows users to sort articles by votes, comments, or publication date, giving them a customized view.

Project Purpose




I developed YC News Grabber to improve the user experience on Y Combinator's news section. Unlike the original platform, which only sorts articles by rank, my YC News Grabber allows users to sort them by votes, comments, or publication date. This makes it easier to find relevant and timely content based on personal preferences.

Web Stack and Explanation




  • Real-Time Data Collection: Continuously gather and update articles from Y Combinator's news page to ensure users have access to the most current information.

  • Customizable Sorting: Offer users multiple sorting options beyond rank, such as by vote count, comment count, or publication time.
  • User-Friendly Interface: Develop an intuitive and responsive web interface that allows users to interact with the data easily, leveraging technologies like Flask, HTML , CSS and Bootstrap.
  • Backend Efficiency: Utilize Python, BeautifulSoup and Pandas for efficient data scraping, processing and storage, ensuring the system runs smoothly and reliably.

What I Have Learned




Developing YC Live Scraper Web App provided valuable insights into web scraping, data manipulation, and web development. I learned the importance of selecting the right tools for each task, such as using BeautifulSoup for efficient data extraction and Pandas for flexible data handling and sorting. Integrating these tools within a Flask-based web application highlighted the challenges of combining backend logic with frontend presentation. Additionally, working with Jinja2 taught me how to dynamically render data-driven content, enhancing the user experience and interactivity of the web app.
